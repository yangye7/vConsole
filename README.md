## vConsole

为方便在`H5`调试，扩展了`Network`模块，为其增加了复制`curl`功能。

![例子](https://pic2.imgdb.cn/item/6454bec90d2dde5777276cba.png)


## 安装

```bash
$ npm install @zj/vconsole
```

## 使用

```javascript
import VConsole from '@zj/vconsole';

const vConsole = new VConsole();
vConsole.destroy();
// 同原文档使用方式一样
```
